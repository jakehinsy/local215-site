# Local 215 Website

Public-facing website for Milwaukee Firefighters IAFF Local 215.

## Purpose
This site provides:
- Public information about Local 215
- Links to the union store and foundation Square site
- Community and charitable foundation information
- A clean public homepage for mfdlocal215.org
