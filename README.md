# Local 215 Website

Public-facing website for Milwaukee Firefighters IAFF Local 215.

## Purpose
This site provides:
- Public information about Local 215
- Links to the union store (Square)
- Community and charitable foundation info
- Entry point to the Platoon app (members)

## Stack
- Next.js
- Vercel (hosting)
- Square (store)

## Domain
https://mfdlocal215.org
